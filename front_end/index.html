Overview

The main aim of the project is to provide a tool where user would be able to upload an image and the software would predict words contained in the image. There are theree parts of the project:

  • denoise the image using machine learning
  • predict the words in the image via deep learning
  • show the found words to the users

    • Denoise the image

    Documents are often dirty, they could be crumpled, have stains, old or have scan marks. This pose a challege for the computer to envision it. In this part of the project, I used machine learning to denoise the image. I have used two datasets(dirty and clearn version of same image) from Kaggle to extract features and then trained a random forest regressor to denoise the dirty documents.

    Predict words with Deep learning

    In this componets of my project, I have used OpenCV for preprocessing the image and then used EAST(An efficient and Accurate Scene Text Detector) pre-trained deep learning model to detect the words in the image. I have also taken care of text if it's slightly rotated or slanted.

    User interface

    User interface is the coponent of the project which string together the denoising and detection componet together along with the user iterface. Users are able to upload an image, which is denoised in turn, and fed into the EAST deep learning model to detect words contained in the image. Once the EAST model is finished with detection, the resulsts are shown to the user.
